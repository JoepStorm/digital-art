## LLM-Evolved Simulations
[This website](https://joepstorm.github.io/digital-art/viewer.html) displays visualizations from simulations. Based on an initial setting and very simple rules, an simple simulation is created. A screenshot of the simulation, along with the code, is given to an LLM with the instruction to add a single feature based on a given theme. This process is repeated many times, showing an evolution of increasing complexity. Some examples are given below, click on any image to watch the corresponding simulation:
